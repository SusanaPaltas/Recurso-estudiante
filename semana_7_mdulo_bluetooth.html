SEMANA 7: MÓDULO BLUETOOTH

Objetivo de aprendizaje

Desarrollar habilidades tecnológicas mediante la creación de una aplicación móvil en App Inventor que permita el control de dispositivos electrónicos a través del módulo Bluetooth.

¿Qué es el modulo Bluetooth?

Es una tecnología que actúa como una interfaz que ayuda a la conexión inalámbrica Bluetooth de bajo consumo de dos dispositivos y establece un protocolo para la comunicación de datos entre los dispositivos (Mucientes, 2021).

Puertos y su función

Puerto Función
STATE indica el estado de conectividad del módulo Bluetooth 
RXD Los datos en serie que recibe RXD se transmiten a través de un protocolo inalámbrico por el módulo Bluetooth.
TXD Mientras está conectado con la unidad del microcontrolador. TXD transmite en serie la información recibida por el módulo Bluetooth, a través del pin TXD.
GND Utilizado para conectar a tierra el módulo. Conecta este pin para conectar a tierra el módulo.
VCC 5V está conectado al pin VCC
KEY Se utiliza para cambiar entre el comando AT del módulo Bluetooth y los modos predeterminados. 

Fuente: Mucientes (2021).
