Ejemplo

Movimiento de servomotor a 45 grados

Esquema de referencia 

Código fuente:

#include <Servo.h> // Incluimos la librería Servo

Servo miServo; // Creamos el objeto Servo

void setup() {
miServo.attach(6); // Conectamos el servo al pin 6
}

void loop() {
miServo.write(45); // Mover a 45 grados
delay(1000); // Esperar 1 segundo

miServo.write(0); // Volver a 0 grados
delay(1000); // Esperar 1 segundo
}
