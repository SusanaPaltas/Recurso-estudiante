Actividad

Crea y modifica tu propio semáforo

¡Es hora de poner en práctica lo aprendido!

Replica y simula: En Tinkercad, crea un nuevo circuito. Conecta los LEDs y el buzzer a los pines correspondientes (Led Rojo al pin 13, Led Amarillo al 12, Led Verde al 11, y Buzzer al 10). Copia y pega el código proporcionado a continuación. ¡Inicia la simulación y observa cómo funciona tu semáforo!

const int ledRojo = 13;
const int int ledAmarillo = 12;
const int ledVerde = 11;
const int buzzer = 10;


void setup() {
  // Configurar pines como salidas
  pinMode(ledRojo, OUTPUT);
  pinMode(ledAmarillo, OUTPUT);
  pinMode(ledVerde, OUTPUT);
  pinMode(buzzer, OUTPUT);
}


void loop() {
  // Luz verde encendida (peatones pueden cruzar)
  digitalWrite(ledVerde, HIGH);
  // Buzzer emite sonidos intermitentes
  for (int i = 0; i < 5; i++) { // El buzzer suena 5 veces
    digitalWrite(buzzer, HIGH);
    delay(300);
    digitalWrite(buzzer, LOW);
    delay(300);
  }
  digitalWrite(ledVerde, LOW); // Apaga la luz verde


  // Luz amarilla (precaución)
  digitalWrite(ledAmarillo, HIGH);
  delay(2000); // 2 segundos
  digitalWrite(ledAmarillo, LOW);


  // Luz roja (peatones deben esperar)
  digitalWrite(ledRojo, HIGH);
  delay(5000); // 5 segundos
  digitalWrite(ledRojo, LOW);
}

Montaje Físico: Haciendo uso del kit didáctico de Robótica, descarga el código desde Tinkercad. Abre el IDE de Arduino (el programa para cargar código a la placa), pega el código y cárgalo a tu placa Arduino. ¡Conecta los componentes físicos y mira tu semáforo encender!

Reto

Modifica tu Semáforo: Ahora que dominas lo básico, es tu turno de modificar el código de tu semáforo para cumplir con las siguientes condiciones:

  • La luz verde debe durar 3 segundos.
  • La luz amarilla debe durar 1 segundo.
  • La luz roja debe durar 4 segundos.
  • El buzzer debe sonar solo 2 veces (en lugar de 5) cuando la luz verde esté encendida.

¡Experimenta con los valores de delay() y el bucle for() para lograrlo!
