SEMANA 3: SERVOMOTORES

Objetivo de aprendizaje

Diseñar y programar el funcionamiento de servomotores en la simulación virtual mediante Tinkercad y de manera física, logrando que estos se muevan en ángulos determinados.

Uso de servomotores con Arduino 

Un servomotor o comúnmente llamado servo, es un motor DC con la capacidad de ubicar su eje en una posición o ángulo determinado (Ortiz y Vásquez, 2022).

Funciones de la librería servo de Arduino

Código  Función
attach(Pin) Establece el pin indicado en la variable servomotor.
attach(Pin,min,max) Establece el pin indicado en la variable servo, considerando min el ancho de pulso para la posición 0° y máx. el ancho de pulso para 180°
write(angulo) Envía la señal correspondiente al servo para que se ubique en el ángulo indicado, ángulo es un valor entre 0 y 180°.
writeMicroseconds(tiempo) Envía al servo el ancho de pulso=tiempo en microsegundos. Ej: servo.writeMicroseconds(1500);
read() Lee la posición actual del servo, devuelve un valor entre 0 y 180.
attached(Pin) Verifica si la variable servo está unido al pin indicado, devuelve true o false.
detach(pin) Separa la variable Servo del pin indicado.

Fuente: Ortiz y Vásquez (2022)
